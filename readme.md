# CI-API-DSM-2026

Projeto desenvolvido para a disciplina de CI/CD.

## Tecnologias

* Python
* Flask
* Git
* GitHub

## Como executar

### Instalar dependências

```bash
pip install -r requirements.txt
```

### Executar aplicação

```bash
python app.py
```

## Endpoints

### GET /

Retorna uma mensagem de funcionamento da API.

### GET /status

Retorna o status da aplicação.
